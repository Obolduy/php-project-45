### Brain Games

#### Минимальные требования

- PHP >= 8.1
- Composer >= 2.0
- Расширение `readline`

#### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Obolduy/php-project-45.git
   cd php-project-45
   ```
2.	Установите зависимости:
   ```bash
   make install
   ```

#### Запуск
Возможные команды для запуска:
   ```bash
   make brain-even
   ```
   ```bash
   make brain-calc
   ```
   ```bash
   make brain-gcd
   ```
   ```bash
   make brain-progression
   ```
   ```bash
   make brain-prime
   ```
